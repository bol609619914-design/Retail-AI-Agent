# Changelog

本文档记录 `Retail-AI-Agent` 的重要迭代。

## [2026-04-09]

### 新增

- 新增联网产品发现能力，后端可结合用户当前诉求搜索更贴近的官方产品页，并提取标题、链接与产品图。
- 新增真实品牌单品与官网素材，推荐卡片支持跳转官方页面查看。

### 优化

- 将推荐流程调整为“本地精选库 + 实时联网候选”双来源结合，再交给顾问式 AI 做最终判断。
- 重做首页头部字标，只保留 `Retail AI Agent`，并统一为浅绿主题的品牌字感。
- 重做右侧推荐区视觉结构，使其更接近左侧 AI 对话气泡的语言：白色独立气泡、深色正文、偏暖层次与更明显的阴影。
- 收紧右侧标签气泡宽度，使 `空间 / 氛围 / 功能` 只包裹文字本身。
- 提升左右正文和推荐区字色对比度，减少页面整体“发灰、发虚”的感觉。
- 更新项目截图，使 README 与当前界面保持一致。

### 修复

- 修复前端多个组件中的中文乱码，包括输入框占位文案、推荐卡片标签、状态提示与推荐区标题。
- 修复右侧推荐卡片与上一轮结果串台的问题，确保新一轮对话会清空旧推荐并显示当前结果。
- 修复模式提示与推荐区文案在不同开发进程下未正确热更新的问题。

## [2026-04-08]

### 新增

- 接入更像顾问的两段式后端流程：先做偏好判断，再生成最终回复。
- 新增 `backend/app/services/consultant_planner.py`，用于决定当前该追问还是推荐。
- 新增 `backend/app/services/web_catalog.py`，支持按用户诉求联网抓取官方产品页，并提取产品标题、官网链接与 `og:image`。
- 推荐卡片新增官网入口，可直接跳转到真实产品页。

### 优化

- 重写 `backend/data/products.json`，将原型产品库替换为真实品牌单品，并补充品牌、价位、材质、工艺、空间标签、搭配建议与官网链接。
- 重写 `backend/app/services/recommendation.py`，修复乱码并提升中文偏好提取与推荐命中逻辑。
- 重写 `backend/app/main.py`，清理中文 prompt 与流式逻辑，确保 `/chat` 返回稳定的中文 SSE 响应。
- 前端首页视觉重新分层：背景、主对话区、推荐侧栏、输入区和推荐卡片的色阶被拉开，不再整页一个色调。
- 提升左侧对话区与右侧推荐区文字对比度，统一到更清晰的正文色阶。

### 修复

- 修复上一轮推荐残留导致右侧卡片不更新的问题。
- 修复后端旧进程和热更新不稳定导致的本地调试混乱。
- 修复 PowerShell 中文测试时的编码干扰问题，改为以 UTF-8 / Unicode 方式验证接口。

## [2026-04-07]

### 新增

- 新增中文主导的聊天首页与顾问式推荐体验。
- 新增结构化推荐卡片，展示产品摘要、偏好命中点、适用场景与替代说明。
- 新增 `/chat` 的 SSE 流式响应，支持逐段渲染顾问回复。
- 新增无 Key 场景下的本地 mock 顾问模式，方便演示与联调。

### 优化

- 将对话节奏调整为更接近线下顾问的三段式：先理解空间，再收拢氛围或功能，最后推荐单品。
- 扩展产品数据结构，加入预算层级、风格标签、空间标签与搭配说明。
- 拆分首页前端组件，降低 `pages/index.vue` 的耦合度。
- 优化 README 结构与项目展示信息。

### 修复

- 修复产品库路径与 UTF-8 BOM 读取问题。
- 修复中文检索过于保守导致推荐卡片难以触发的问题。

## [2026-04-06]

### 新增

- 初始化 Nuxt 3 + Tailwind CSS 前端项目。
- 初始化 FastAPI 后端框架。
- 新增本地 JSON 产品库与基础商品搜索接口 `/api/products/search`。
- 新增 GitHub 展示用 README、截图与 MIT License。
